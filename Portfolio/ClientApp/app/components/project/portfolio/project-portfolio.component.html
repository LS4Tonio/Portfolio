Mon CV en ligne

Ce projet est un prérequis pour l'obtention de mon diplôme d'expert en ingénierie du logiciel, RNCP niveau 1, délivré par l’école IN’TECH du groupe ESIEA. Ce site vous montre les différentes compétences et projets qui me sont importants.

Ce site a été développé entièrement en utilisant les technologies ASP .NET Core pour la partie serveur et Angular (version 4) pour la partie cliente.
C’est l’occasion pour moi de me former sur la nouvelle version d’Angular. Ce framework Angular, au contraire de son précurseur, AngularJS, a été écrit en TypeScript.
Le TypeScript est un langage développé par Microsoft qui permet d’écrire du code tel qu’on le ferait avec le C#. Ce code doit être ensuite transformé en JavaScript pour que le navigateur le comprenne.

La raison principale pour avoir développé ce site web, c’est l’envie de démontrer mes compétences à travers un exemple concret qu’est mon portfolio.
L’autre raison, plus ludique, est celle de l’apprentissage d’une nouvelle technologie et sa mise en application. Travailler sur de nouvelles technologies et s’amuser en la pratiquant est l’un de mes principaux moteurs d’exécution.
