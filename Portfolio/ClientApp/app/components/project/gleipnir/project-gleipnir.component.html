Mon premier jeu en client lourd

Gleipnir est une application lourde pour un jeu de gestion développé en C# avec le framework .NET et avec la technologie Windows Forms. Il a été réalisé dans le cadre d’un projet d’étude du semestre 3 à IN’TECH.

Le joueur incarne le rôle d’un dieu qui doit participer au bon développement d’un village. Le but est de garder l’économie, la foi, le bonheur et la nourriture à niveau pour éviter de perdre.

Ce jeu est le premier projet réalisé avec un langage orienté objet. Nous étions trois pour développer du jeu sur une période de trois mois. Nous avons tous fait, de la réalisation des pages utilisateurs à la logique de gestion des composantes du jeu.

Nous avons établi des spécifications pour définir le périmètre de développement. Puis j’ai été en charge de développer l’interface graphique ainsi que les actions utilisateurs.

Maintenant, avec les compétences que j’ai acquis en programmation orienté objet, je referais le projet en utilisant des technologies plus adaptées pour développer un jeu.
