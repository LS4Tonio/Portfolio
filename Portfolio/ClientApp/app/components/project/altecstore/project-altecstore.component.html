Un store applicatif

Altec Store est un ensemble d’applications développées, ou intégrées, dans un store applicatif développé pour l’occasion. Ce store d’application a été réalisé à la demande d’Air Liquide. Le client voulait que le store applicatif, appelé store, soit administrable afin de pouvoir gérer les différentes versions des logiciels et de leurs données pour les applications métiers.

Le but était donc de développer un client lourd servant de store qui installe, met à jour et désinstalle des applications en fonction de l’utilisateur connecté au PC. Il fallait créer le système de centralisation et gestion des versions des applications à l’aide d’une base de données. Puis mettre en place le système d’installation, mise à jour et désinstallation automatique. Finalement, ce store applicatif a pour but de pouvoir exécuter les applications à la manière de la liste d’applications d’un smartphone.

Les applications lourdes ont été développée en C# et WPF, Windows Presentation Foundation, en utilisant le framework .NET. En ce qui concerne les bases de données, celles-ci ont fonctionnent avec SQL Server. Tous les sites Web d’administration ont été développés en ASP .NET. Afin de mieux séparer les administrations de chaque application métier, nous avons développés des services WCF, Windows Communication Foundation. Les mécanismes du store utilisent le Batch pour installer, mettre à jour ou désinstaller une application.

J’ai participé au développement de ce projet en intervenant à la fois sur le développement d’applications métiers à ajouter dans le store et à la fois sur le développement du store. Puis j’ai aussi travaillé sur le site d’administration du store et enfin, j’ai mis en place les mécanismes d’installation, mise à jour et de désinstallation du store.

Au cours de ce projet j’ai dû apprendre le JavaScript couplé avec la libraire jQuery, le Batch pour les mécanismes du store applicatif, le WPF ainsi que le fonctionnement du WCF. Et j’ai aussi appris à travailler avec une base de données SQL Server.
Ce projet m’a beaucoup apporté par sa complexité et les technologies diverses et variées sur lesquelles j’ai pu monter en compétence.
