Un outil de monitoring

Lors de mon stage dans la DSI, Direction des Systèmes d'Information, de France Media Monde, on m’a donné comme projet de développement un outil de monitoring pour visualiser l’état des flux d’enregistrement de la TV & radio. Cet outil doit permettre à un technicien de voir l’état des flux à tout moment.

Cet outil est une application Web développée en ASP .NET pour le côté serveur et JavaScript avec la librairie jQuery pour la partie client.

Je me suis basé sur l'architecture d'une autre application réalisée pour développer tout le site avec l'aide d'un jeu de données de test.

Ce projet, d'une durée de un mois, a été le premier projet que j'ai réalisé avec l'ASP .NET ainsi que ma première expérience avec le JavaScript.
Ce projet est une réussite car il a été lancé en production à la fin du développement que j'ai réalisé.
