Un outil de monitoring

Lors de mon stage dans la DSI de France Media Monde, on m’a donné comme projet de développement un outil de monitoring pour visualiser l’état des flux d’enregistrement de la TV & radio. Cet outil doit permettre à un technicien de voir l’état des flux à tout moment.

Cet outil est une application Web développée en ASP .NET pour le côté serveur et JavaScript avec la librairie jQuery pour le client.
