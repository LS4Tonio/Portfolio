Le scripting Web

Le JavaScript est un langage de programmation et de script pour les navigateurs Web. Il permet la manipulation d’une page Web afin de la rendre dynamique, c’est-à-dire sans rechargement requis.

J’ai appris le JavaScript que tard dans ma scolarité : j’ai commencé à partir du semestre 5, début de la troisième année. J’ai très vite progressé lorsque je suis arrivé en entreprise pour le stage de 6 mois du semestre 6.
Je me suis familiarisé avec la syntaxe et les concepts à travers les projets que l’on m’a donné.

Lorsque je me suis formé sur l'AngularJS j'ai dû approfondir mes connaissances du langage pour pouvoir maitriser le framework. J’ai franchi le cap de la manipulation de page HTML pour pouvoir créer une application Web côté client avec des appels de données sur une API ou pour la gestion d’une navigation dynamique.

Depuis, tous les développements d'application Web que je fais utilisent du JavaScript pour rendre les interactions dynamiques et fluides. Je n’envisage plus de faire un site Web en utilisant que du HTML et du CSS.
