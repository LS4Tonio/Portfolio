Le rendu pour le navigateur

Votre navigateur ne comprend pas beaucoup de chose pour vous afficher une page Web. Il ne comprend que le HTML, le CSS et le JavaScript.

Le HTML, pour HyperText Markup Language ou Langage de Balisage Hyper Texte, est le langage utilisé pour l’affichage de tous les éléments d’un site Web. C’est ce langage qui permet de dire « Ceci est titre » ou « Ceci est un bouton ».

Le CSS, pour Cascading Style Sheets ou Feuilles de Style en cascade, met en forme tous les éléments HTML. Par exemple un bouton est par défaut rectangulaire, si vous le voulez rond, le CSS permet de configurer le rendu de votre bouton.
Il permet également de pouvoir gérer facilement le rendu d'une page Web sur différents appareils comme une tablette ou un téléphone.

Ces deux langages sont les prérequis pour créer des sites et applications Web. Ce sont des technologies que je pratique depuis mon entrée à IN’TECH.

Sans elles, mon portfolio ne serait pas comme vous le voyez à présent. J’ai placé et stylisé chaque ligne ou élément visible de la page que vous lisez.
