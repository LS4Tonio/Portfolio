Pour du Web Dynamique

AngularJS est un framework JavaScript pour développer des applications Web dynamiques. Il est écrit en JavaScript, langage de Scripting utilisé dans les navigateurs Web et repose sur l’utilisation de jQuery, une librairie JavaScript pour la manipulation des éléments d’une page Web. Ce framework est sorti en 2009.

J'ai travaillé quotidiennement pendant plus d'un an avec AngularJS. J'ai pu parcourir un bon nombre de concepts et fonctionnalités pour créer des applications Web dynamiques.
N'hésitez pas à regarder les projets dans la section Projets pour des exemples de réalisations avec AngularJS.

La dernière version d'AngularJS, appelée Angular, apportant de nombreux changement et de grandes améliorations est sorti en 2016.
La principale différence concerne le langage d'écriture du framework. Angular est écrit en TypeScript, un langage permettant d'avoir un code JavaScript plus optimisé et performant.
De plus, plusieurs concepts ont été revus pour permettre de réaliser des applications sous forme de Web Components, un bloc permettant de définir une fonctionnalité et réutilisable.

Le site que vous parcourez est réalisé en Angular pour monter en compétences sur cette nouvelle version du framework.
Pour vous montrer le dynamisme de l'application, chaque lien du site ne fait changer que le contenu entre le haut et le bas de la page Web affichée.
Chaque bloc dans la page correspond à un composant qui peut être réutilisé n'importe où ailleurs dans l'application.
