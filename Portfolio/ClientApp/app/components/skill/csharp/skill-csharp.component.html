Le langage orienté object du .NET

Le Framework .NET est un ensemble de technologies développé par Microsoft et s’appuyant sur les systèmes d’exploitations de celui-ci. Il met à disposition un environnement d’exécution pour les langages compatibles CLI (Common Language Infrastructure).

Le C# est un langage de programmation orienté objet. Il a été introduit par Microsoft pour pouvoir développer en s’appuyant le Framework .NET.
Il est l’équivalent du Java par ses concepts et sa syntaxe.

Il permet de développer à la fois des applications lourdes, des applications Web et des services.
J'ai fait majoritairement des applications Web, en ASP .NET, dont toute la partie serveur est écrite en C#.

Si vous voulez en savoir plus, n'hésitez pas à aller voir l'introduction au C# et le framework .NET.
