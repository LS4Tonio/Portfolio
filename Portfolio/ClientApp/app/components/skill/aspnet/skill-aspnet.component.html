Le Back-End Web de Microsoft

C’est l'une des technologies du framework .NET, un ensemble de technologie mises à disposition aux développeurs. Elle permet la création d’applications Web.
L’ASP.NET se compose d’une partie back, ou serveur, et d’une partie front appelée aussi client.
La technologie inclue de nombreux mécanismes tels que des mécanismes d’authentification, d’API, de formatage JSON et bien d’autres techniques utilisées aujourd’hui dans les applications Web.

La technologie a un bel avenir devant elle puisque une nouvelle version, basée sur le nouveau framework .NET Core, est sortie en 2016.
Microsoft a mis à jour la manière de créer une application Web pour correspondre aux pratiques courantes de la création d'application Web, tel que NodeJS ou ReactJS.

J’utilise cette technologie, dans sa dernière version, pour le portfolio que vous lisez. Le back met à disposition une API pour les données et le front s’occupe de gérer la navigation et l'affichage. Je sais à la fois mettre en place un serveur, créer et rendre un API disponible. Je sais configurer l’authentification et toute la chaine de traitement du serveur.
